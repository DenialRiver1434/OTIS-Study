# OTIS Study Archive
This is a collection of the solution writeups I have completed over the hundreds of hours I have spent
with Evan Chen's OTIS (Olympiad Training for Individual Study) program, which I highly recommend.

### Information Redacted

To protect OTIS' handout security, the example problems, problem statements, handout names, and lecture
handouts are omitted. I uploaded these solutions to a repository for archival storage purposes since OTIS
has official solutions and solutions have also been posted to AoPS by other users.
